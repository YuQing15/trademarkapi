Class 9
Application software for mobile phones and media devices incorporating pre-recorded sign language videos for mobile phones; plugin software; computer software, being a browser extension to enable users to watch simultaneous transmission of sign language content alongside written text in a web page; virtual and augmented reality software for use in mobile devices; downloadable software in the form of desktop, mobile, tablet, laptop and computer applications; embedded operating software; business technology software; data synchronisation software; downloadable software for multi-party access to sign language content; communications software; computer software for scanning images, barcodes and audiovisual content and delivering corresponding sign language content about that media to users; software in the nature of a mobile application for use with portable handheld electronic communication devices for accessing, retrieving, reproducing and transmitting sign language content; content access software enabling the conversion of written content into sign language; application software for streaming content via the internet; computer software for wireless content delivery; software for use in downloading, transmitting, receiving, editing, extracting, encoding, decoding, playing, storing and organising data including video data; pre-recorded videos and media featuring sign language content; software enabling the translation of written content into sign language content and the delivery of that content to a user’s mobile device and presentation thereon; recorded content; video and data recordings; computer software to enable the provision of sign language content via communications networks; downloadable video and image files; electronic publications; all for use in comprehension, analysis, translation or manipulation of data into sign language to increase content accessibility for the deaf and hearing-impaired community.
