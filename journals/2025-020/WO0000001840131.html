Class 42
Providing temporary use of online non-downloadable software for collecting, compiling, processing, transmitting, and disseminating composable profiles and user identities, geolocation data, social network data, calendar data, user data from social network services, subscription data, social graph data, and visitation graph data; providing temporary use of online non-downloadable software for social networking and creating, sharing, disseminating, posting and transmitting messages, photos, videos, and other media across the internet and multiple operating systems; providing temporary use of non-downloadable web-based decentralized applications (DApps) comprising a blockchain-based decentralized social graph protocol designed to enable user-owned modular content using non-fungible tokens; providing temporary use of online non-downloadable software that allows users to create and manage digital representations of non-fungible tokens to facilitate the portability of user-generated and user-owned profiles and content; software-as-a-service (SAAS) services featuring open-source software and tools in the nature of software development tools for building a platform for products and services that run according to a cryptocurrency protocol that coordinates with a distributed social graph.
Class 45
Online social networking services launched by individuals using an open-source decentralized social graph compatible with blockchain-based wallets and non-fungible tokens.
