Downloadable mobile and desktop software applications for managing user authentication, encrypted CV storage, and role-based access to business workflows; downloadable software for completing training, attending scheduled events, and tracking task or time data linked to a hosted SaaS platform; downloadable applications for contractors and users to manage personal profiles, submit self-service HR requests, and maintain employment-related data; downloadable software for viewing assignments, updating user records, and managing client interactions. .
Class 42
Software as a service (SaaS) featuring software for managing user identities, access permissions, and secure user profile data; SaaS platform for workflow automation, employee and contractor onboarding, and enterprise lifecycle management through to offboarding; cloud-based platform for storing, encrypting, and selectively sharing user CVs and profiles with employer organisations based on user consent; SaaS for user-initiated profile updates including name, photo, and personal details, routed through employer review and approval workflows; software platform for orchestrating synchronised updates across third-party enterprise systems connected to a user’s profile; SaaS for scheduling, managing, and tracking events, training sessions, and client engagements; software for businesses to manage contractor and employee provisioning, project assignments, time tracking, and asset or IT equipment allocation; SaaS for managing internal asset inventories and tracking device assignments; platform as a service (PaaS) incorporating artificial intelligence (AI) models for user profiling, recommendation engines, and operational optimisation; provision of APIs and software plugins for integration with external enterprise systems, including procurement, finance, HR, and identity platforms; orchestration of identity and user status changes across connected systems based on event triggers such as onboarding, offboarding, reassignment, or profile updates. .
