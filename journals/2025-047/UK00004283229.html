Class 9
Computer software for distribution of tickets; event management software; software for managing customer orders; computer software and downloadable software applications relating to the organising, promoting, hosting, producing, booking, reservation and ticketing of events; software and downloadable software applications for providing access to events; software for seat reservation; computer software to enable event organisers to sell tickets, monitor event attendance data, and track event sales; downloadable software for processing payments associated with event ticket reservations and sales.
Class 35
Booking agency services; business management of entertainment venues; administrative processing of purchase and sales orders; administrative processing of sales and orders of tickets for providing access to events; sales promotion for others; management of event ticketing for others; promoting events for others; provision of an online and offline marketplace for the sale and purchase of tickets; information, advice and consultancy services relating to all the aforesaid services.
Class 41
Arranging, organising, producing and hosting events; event management services; box office services, namely selling tickets and managing access to events; ticketing and event booking services; booking, seating, ticket reservation and sales services relating to events; information and ticket purchasing agency services for venues; creating, issuing, validating or checking tickets and passes for access to events; information, advice and consultancy services relating to all the aforesaid services.
Class 42
Provision of a non-downloadable software for event management and event ticketing; platform as a service (PaaS) featuring software for event organisers to sell tickets to events, manage bookings, promotion, manage attendee and event information; platform as a service (PaaS) featuring computer software platforms for use in connection with selling event tickets and event merchandise; software as a service (SaaS) services featuring non-downloadable software for managing events, promoting events, advertising events, marketing events, reporting on event sales, managing inventory and communicating with event ticket holders; software as a service (SaaS) services featuring non-downloadable software for managing event ticketing, namely, providing to event organisers and event ticket sellers, the ability to manage their ticketing sales operations; software as a service (SaaS) services featuring software for processing payment and tracking data related to event tickets; information, advice and consultancy services relating to all the aforesaid services.
