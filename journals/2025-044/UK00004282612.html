VeriClear

Class 9
Downloadable mobile applications for capturing, verifying and sharing photographic and video evidence; downloadable software for project management, work verification, and payment processing; computer software for use in proof-of-work validation, digital verification, and milestone tracking; downloadable applications for use in providing timestamped and geotagged images and videos; downloadable computer programs for use in verifying completion of tasks, projects, or services.
Class 36
Payment processing services; escrow services for electronic payments; financial transaction verification services; electronic transfer of funds linked to verified project milestones.
Class 42
Software as a service (SaaS) for capturing, verifying and sharing photographic and video evidence; providing online non-downloadable software for work verification, project management, and automated payment processing; providing temporary use of online non-downloadable software for use in digital verification and proof-of-work systems; cloud-based verification platforms using geotagged and timestamped media; design and development of computer software for use in authentication and verification.
