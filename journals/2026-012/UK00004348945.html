Downloadable mobile application software for providing healthcare services, namely, software for monitoring and tracking neurostimulator therapy for the treatment of chronic inflammatory and autoimmune disorders; Downloadable mobile application software for use in adjusting the settings of associated medical devices used to treat chronic inflammatory and autoimmune disorders through neurostimulator therapy; Downloadable mobile application software for providing users with status information and notifications regarding associated medical devices, namely, operational status, firmware status, battery level, device expiration date, and software versions; Downloadable mobile application software for use in viewing, suspending, rescheduling, and controlling doses of neurostimulator therapy administered by associated medical devices; Downloadable mobile application software for providing manuals, guides, and videos with information on how to use associated medical devices used to treat chronic inflammatory and autoimmune disorders through neurostimulator therapy; Downloadable mobile application software providing users with a digital patient ID card; Downloadable mobile application software for wireless data communication for receiving, processing, transmitting, aggregating, analyzing, and displaying information relating to neurostimulator therapy; Downloadable mobile application software to enable the transfer of data from medical devices to clinical personnel and healthcare professionals; Downloadable mobile application software for providing users with information about chronic inflammatory and autoimmune disorders; Downloadable mobile application software for tracking chronic inflammatory and autoimmune disorder activity metrics and transmitting that data to healthcare providers.
