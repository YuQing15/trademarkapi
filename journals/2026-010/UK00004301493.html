RentReward

Class 9
Downloadable software; mobile applications; software for tracking rent payments; software for verifying financial transactions via Open Banking; software for awarding reward points; software for credit score improvement; software for property management; application software enabling financial data processing; software for tenants and landlords to manage rental payments.
Class 36
Financial services; verifying rent payments using Open Banking data; rental payment reporting services; credit score improvement services; provision of financial information; processing and verification of financial transactions; electronic payment verification; financial data analysis relating to rental payments; financial reward programmes; providing information and advice relating to financial creditworthiness based on rent payment behaviour.
Class 42
Software as a service (SaaS); Providing online, non-downloadable software; Platform as a service (PaaS); hosting of software platforms; providing temporary use of non-downloadable software for tracking rent payments, issuing rewards, and supporting credit score improvement; software development services; design and development of financial technology platforms; data-as-a-service (DaaS); providing online non-downloadable software for landlords and tenants to manage rental information and verify on-time payments.
