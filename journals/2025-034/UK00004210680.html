.rclx

Class 9
Downloadable computer software for creating, reading, writing, encoding, compressing, and semantically structuring digital content in proprietary computer file formats [including the .rclx format]; downloadable software libraries encoding, decoding and validating proprietary computer file formats [including the .rclx and .rclx3, formats and their encrypted variants] ; downloadable tools for binary transformation, semantic fidelity scoring, and regeneration verification; downloadable software wrappers and data structures for use with large language models (LLMS) and artificial intelligence (AI) platforms; downloadable digital container software for lossless and lossy compression with cryptographic binding; downloadable file format utilities for use in machine learning pipelines, AI assistants, and web-based interfaces; downloadable software for forensic tagging, trust-layer integration, and regeneration control across semantically structured data files.
Class 42
Provision of non-downloadable software for the encoding, compression, and certification of semantic computer file formats; software as a service (SaaS) featuring tools for AI validation, semantic regeneration, and fidelity scoring of proprietary computer files formats [including the .rclx format and its variants]; provision of cloud-based tools for managing certified and non-certified proprietary computer file format structures [including the .rclx format]; development and maintenance of compression frameworks and semantic trust protocols; hosting of AI-compatible file wrappers and runtime semantic scoring systems; provision of online non-downloadable software infrastructure for trust-layer deployment, regeneration gating, and integrity validation of compressed data streams in real-time AI and web environments. .
