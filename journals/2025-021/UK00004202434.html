CrowdMaps

Class 9
Downloadable software for displaying location-based information; downloadable mobile applications providing interactive maps and real-time data visualisation; software for tracking user movement and venue density on digital maps; downloadable software for showing user activity and engagement across geographic locations; mobile software for social venue mapping, live crowd analysis, and proximity-based recommendations; software integrating geolocation with venue discovery platforms.
Class 35
Business data analytics relating to footfall, venue activity, and geographic behaviour; marketing services powered by geolocation and user proximity data; provision of business intelligence for venue and event operators through live map visualisations; analysis of crowd behaviour and user flow for marketing and promotional purposes; data aggregation services for location-based insights and venue mapping tools.
Class 42
Software as a Service (SaaS) for map-based crowd monitoring and user location tracking; hosting of interactive map platforms for real-time activity insights; development of geolocation-based mapping platforms incorporating crowd activity patterns and real-time user behaviour tracking; platform infrastructure for venue visibility, density mapping, and user activity overlays; backend system development for live map features used across social, hospitality, and event-based platforms.
