GyroFX

Class 9
Downloadable software and hardware systems for recording, analysing, and visualising aerobatic flight data; flight performance analysis software for use in aviation training and competition; systems for telemetry, motion tracking, and flight path optimisation; electronic hardware and software for integrating gyroscopic and accelerometer data into flight analytics systems; downloadable mobile applications for real-time manoeuvre diagnostics and post-flight review; integrated platforms for flight behavior visualisation, spatial mapping, and pilot coaching tools.
Class 42
Providing temporary use of non-downloadable cloud-based software for for flight performance analysis, pilot training, aviation competitions and aeronautical data visualisation; software as a service (SaaS) featuring platforms for pilot performance review, real-time manoeuvre diagnostics, and spatial mapping; design and development of flight analysis software and telemetry systems; hosting platforms for virtual flight coaching, aviation competition analytics and competition scoring.
